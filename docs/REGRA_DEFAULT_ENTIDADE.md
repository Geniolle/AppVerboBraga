# Regra default de entidade

Toda entidade criada no sistema deve possuir estes campos:

- Estado
- Nº Entidade

## Regra visual

O campo Nº Entidade deve aparecer ao lado do campo Estado.

Exemplo:

Estado: Ativo
Nº Entidade: EXT-000001

## Nomes técnicos

Frontend/Formulário:

- estado
- numero_entidade

Base de dados:

- estado
- numero_entidade

## Exemplos de numeração

- Extrato: EXT-000001
- Tesouraria: TES-000001
- Utilizador: USR-000001
- Música: MUS-000001
- Evento: EVT-000001
- Departamento: DEP-000001
